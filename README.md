## Usage

### Setup Instructions

1. Clone the repository: `git clone https://github.com/your-repo-url`
3. Set up the database: Import the schema from `database/schema.sql`
4. Run the server: `java -jar server.jar`
5. Run the client: `java -jar client.jar`
